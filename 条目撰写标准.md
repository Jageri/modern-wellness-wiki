# 条目完整性标准

每个条目必须满足以下全部条件，否则标记为"⚠️ 待完善"。

---

## 字段结构

| 字段 | 要求 |
|------|------|
| 死亡率影响 | 必填。具体数字 + 来源（如"+30%全因死亡风险，BMJ 2024 Meta分析"） |
| 证据等级 | 必填。⭐⭐⭐⭐⭐ Meta分析/大型RCT / ⭐⭐⭐⭐ 多项研究一致 / ⭐⭐⭐ 单项大型研究 / ⭐⭐ 小样本 / ⭐ 专家意见 |
| 结论 | 必填。1-2句话，直接给答案 |
| 证据 | 必填。至少1条，附原文URL |
| 做 | 必填。至少3条可操作建议 |
| 不做 | 至少2条常见误区（如无可靠来源支撑，不要硬凑）|
| 就医信号 | 至少2条（如无可靠来源支撑，不要硬凑）|
| 参考文献 | 必填。按照论文格式，在正文中标注引用符号[1][2]等，并在文末按论文格式列出文献及其线上链接 |

## 来源要求

- 优先：Lancet / BMJ / NEJM / JAMA / Nature / WHO / CDC，其他权威机构发布也可
- 每条核心数据必须附原文URL
- 无来源的结论标注"⚠️ 待验证"或删除
- 不接受"多项研究表明"这种模糊表述——必须指明是哪项研究
