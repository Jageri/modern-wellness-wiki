# 条目完整性标准

每个条目必须满足以下全部条件，否则标记为"⚠️ 待完善"。

---

## 必填字段

| 字段 | 要求 |
|------|------|
| 死亡率影响 | 具体数字 + 来源（如"+30%全因死亡风险，BMJ 2024 Meta分析"） |
| 证据等级 | ⭐⭐⭐⭐⭐ Meta分析/大型RCT / ⭐⭐⭐⭐ 多项研究一致 / ⭐⭐⭐ 单项大型研究 / ⭐⭐ 小样本 / ⭐ 专家意见 |
| 结论 | 1-2句话，直接给答案 |
| 证据 | 至少1条，附原文URL |
| 做 | 至少3条可操作建议 |
| 不做 | 至少2条常见误区 |
| 就医信号 | 至少2条 |

## 来源要求

- 优先：Lancet / BMJ / NEJM / JAMA / Nature / WHO / CDC
- 每条核心数据必须附原文URL
- 无来源的结论标注"⚠️ 待验证"或删除
- 不接受"多项研究表明"这种模糊表述——必须指明是哪项研究

## 梯队划分标准

| 梯队 | 标准 |
|------|------|
| 🔴 第一梯队 | HR ≥ 1.5 或明确因果关系 |
| 🟠 第二梯队 | HR 1.2 ~ 1.5 |
| 🟡 第三梯队 | HR 1.1 ~ 1.2 |
| 🟢 第四梯队 | HR < 1.1 或间接影响 |
| 🔵 第五梯队 | 预防性/间接措施，无直接死亡率数据 |

HR = Hazard Ratio（风险比），来自Meta分析或大型队列研究。

## 校验流程

1. 实时搜索该主题最新Meta分析/指南
2. 核实具体数字和研究来源
3. 补充遗漏的重要证据
4. 无支持证据的结论标记"⚠️ 待验证"或删除
5. 检查必填字段完整性

## 状态标记

- ✅ 已验证：所有字段完整，来源可靠
- ⚠️ 待完善：部分字段缺失或来源待核实
- ❌ 不合格：缺少关键字段或来源不可靠
