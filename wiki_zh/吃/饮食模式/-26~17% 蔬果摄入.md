# 蔬果摄入

**死亡率影响**: 每天 400-800g 蔬果可降低全因死亡率约 13-26%
**证据等级**: ⭐⭐⭐⭐⭐（大量前瞻性研究和 Meta 分析一致）
**可操作性**: 高

## 结论
每天至少 5 份蔬果（约 400g），其中水果和蔬菜各半。这是全球几乎所有长寿饮食模式的共同特征。更大摄入量（最高约 800g/天）可能带来进一步收益。

## 证据
- **Aune D et al., Int J Epidemiol 2017 (PMID 28338764)**：纳入 95 项前瞻性研究的剂量-反应 Meta 分析。每增加 200g/天蔬果摄入，全因死亡率 RR = 0.90（95% CI 0.87-0.93），CVD 死亡率 RR = 0.92（0.90-0.95），总癌症 RR = 0.97（0.95-0.99）。收益持续至约 800g/天。估计 2013 年全球 560 万人过早死亡可归因于蔬果摄入低于 500g/天，780 万归因于低于 800g/天。
- **Wang X et al., BMJ 2014 (PMID 25073782)**：16 项前瞻性队列研究 Meta 分析。每天每增加 1 份蔬果，全因死亡率 HR = 0.95（0.92-0.98），CVD 死亡率 HR = 0.96（0.92-0.99）。存在约 5 份/天的阈值效应，超过后风险不再进一步降低。
- **Wang DD et al., Circulation 2021 (PMID 33641343)**：2 项美国队列 + 26 项队列 Meta 分析。每天 5 份蔬果 vs 2 份：总死亡率 HR = 0.87（0.85-0.90），CVD 死亡率 HR = 0.88（0.83-0.94），癌症死亡率 HR = 0.90（0.86-0.95），呼吸系统疾病死亡率 HR = 0.65（0.59-0.72）。
- **Bondonno NP et al., Nature Communications 2019 (PMID 31409784)**：丹麦 56,048 人、23 年随访。黄酮类摄入量与全因、CVD 和癌症死亡率呈负相关。最高 vs 最低五分位数组，全因死亡率降低约 17%。保护效应在约 500mg/天时达到平台期。吸烟者和高酒精摄入者获益更显著。
- **Du H et al., Int J Epidemiol 2017 (PMID 28449053)**：中国慢性病前瞻性研究（CKB），462,342 人、~7 年随访。经常吃新鲜水果（≥4 天/周）vs 很少/不吃者：全因死亡率 RR = 0.73（0.70-0.76），即降低 27%。
- 关键营养素：膳食纤维、钾、叶酸、维生素 C、多酚类（黄酮类等）、类胡萝卜素
- 蔬菜 ≠ 土豆：淀粉类蔬菜的保护效果较弱或不显著（Aune 2017）

## 局限性与注意事项
- 所有证据来自观察性研究（前瞻性队列），无法完全排除混杂（健康用户偏倚）
- 癌症死亡率的保护效应弱于 CVD，部分 Meta 分析未达统计显著
- 抗氧化剂补充剂的 RCT 未复制蔬果的保护效果（如 β-胡萝卜素补充剂在吸烟者中增加肺癌风险），提示全食物中的多种成分协同作用
- 土豆等淀粉类蔬菜与绿叶菜、十字花科蔬菜的保护效应差异显著，不应等同对待

## 做
1. 每天至少 5 份：2 份水果 + 3 份蔬菜（1 份 ≈ 80g），可逐步增加至 800g/天
2. 颜色越丰富越好：深绿叶菜、红色番茄、橙色胡萝卜、紫色茄子
3. 优先选本地应季蔬果（营养密度更高）
4. 果汁 ≠ 水果（果汁去掉了纤维，糖分浓缩）
5. 每天一杯茶 + 一个苹果 + 100g 蓝莓 + 100g 西兰花 = 超过 500mg 黄酮类（平台期剂量）
