**条目**: #49 空气污染 / Air Pollution

## 一、核心主张验证

### 1. PM2.5每增加10μg/m³，全因死亡率增加约6%
| 项目 | 内容 |
|------|------|
| **原条目主张** | 约6% |
| **验证来源** | Chen & Hoek, 2020, PMID 32703584（WHO委托全球荟萃分析，N=107项队列研究） |
| **实际数据** | RR=1.08（95%CI 1.06-1.09），即**+8%** |
| **亚太数据** | Oh et al., 2025, PMID 40662655: RR=1.11（95%CI 1.05-1.17），即**+11%** |
| **判定** | ❌ **错误** — 低估了效应量。已修正为8%（全球）/ 11%（亚太） |

### 2. 全球疾病负担研究：空气污染是全球第四大死亡风险因素
| 项目 | 内容 |
|------|------|
| **原条目主张** | 第四大 |
| **验证来源** | GBD 2015（PMID 28408086）: 第五位; GBD 2021（PMC12325274）: 第四位，仅次于高血压、吸烟、不良饮食 |
| **判定** | ✅ **准确**（与最新GBD 2021数据一致），但需注明数据年份 |

### 3. 中国北方冬季雾霾与预期寿命差异显著相关
| 项目 | 内容 |
|------|------|
| **原条目主张** | "中国北方冬季雾霾与预期寿命差异显著相关" |
| **验证来源** | Chen et al., 2013, PMID 23836630（PNAS, 淮河供暖政策准自然实验） |
| **实际数据** | 北方TSP比南方高约184μg/m³（+55%），预期寿命低约5.5年（95%CI 0.8-10.2），每增加100μg/m³ TSP缩短预期寿命约3.0年 |
| **判定** | ✅ **准确**，但原条目描述过于简略 |

### 4. 主要死因：心血管病、呼吸系统疾病、肺癌
| 项目 | 内容 |
|------|------|
| **判定** | ✅ **准确**，与荟萃分析一致。已补充具体病因：缺血性心脏病、脑血管病（中风）、COPD、下呼吸道感染、肺癌 |

### 5. N95口罩建议
| 项目 | 内容 |
|------|------|
| **原条目主张** | "重度污染天戴N95口罩" |
| **验证来源** | Jiang et al., 2021, PMID 34265631（双盲RCT，北京，N=52）：N95组FEV1改善+2-3%，呼吸道氧化应激和全身炎症标志物降低。PM2.5>75μg/m³时获益更大 |
| **判定** | ✅ **有RCT支持**，已补充具体证据 |

## 三、新增内容

### 近因机制
补充PM2.5→死亡的四条近因路径：氧化应激→内皮功能障碍→动脉粥样硬化；肺部慢性炎症；全身性炎症→胰岛素抵抗；表观遗传改变。

### 争议与限制
1. 因果 vs 关联的界限
2. 社会经济混杂因素
3. 剂量-反应曲线形状争议（超线性 vs 非线性IER模型）
4. N95口罩长时间佩戴的呼吸阻力副作用

### 反面证据
- GBD 2015 vs 2021排名变化（第五→第四），需注明时效性
- 亚太地区效应值（+11%）显著高于全球均值（+8%），提示地域差异
- 部分研究认为社会经济因素可能部分解释观察到的关联
